**Steps :** 



1. create an Empty flutter app using :   **flutter create --empty app_name**

2. go to app using : **cd app_name**

3. To test app run flutter application using : **flutter run**

4. In your app folder go to lib and create an dart file : **eg . question1.dart, card.dart etc**

5. just copy paste given code : **Follow the instructions (Optional Question Based):**

    

    **Adding Image:** 

    

    1. create assets folder in your app root folder for Images : **app_name/assets/image.png**

    2. add image in **pubspec.yaml : Like this**

    

    ```

    flutter:

      uses-material-design: true

      assets:

          - assets/image.png

    ```

    

    iii.  use image in your file :

    

    ```

    child: Image.asset(

                  'assets/image.png', // your image path

    ```

    

    iv. run in terminal: **flutter pub get ||** if not working use: **flutter clean** then use **flutter pub get**

    

    v. Run app: **flutter run â†’ select device â†’ Output**

    

    **Adding google fonts:** 

    

    1. In **pubspec.yaml**  file add :

    

    ```

    dependencies:

      flutter:

        sdk: flutter  

      google_fonts: ^6.2.1

    ```

    

    ii.   Import if not already Imported in Question file  : 

    

    **import 'package:google_fonts/google_fonts.dart';** 

    

    iii.  use given font: Like This :

    

    | **Font Name** | **Code Example** |

    | --- | --- |

    | **Roboto** | style: GoogleFonts.roboto(fontSize: 20) |

    | **Open Sans** | style: GoogleFonts.openSans(fontSize: 20) |

    | **Lato** | style: GoogleFonts.lato(fontSize: 20) |

    | **Montserrat** | style: GoogleFonts.montserrat(fontSize: 20) |

    | **Poppins** | style: GoogleFonts.poppins(fontSize: 20) |

    

    iv.  eg . **GoogleFonts.fontName()**

    

    ```dart

    style: GoogleFonts.pacifico( // font name is pacifico

                  fontSize: 28,

                  color: Colors.purple,

                  fontWeight: FontWeight.bold,

                ),

    ```

    

    v.  run in terminal: **flutter pub get ||** if not working use: **flutter clean** then use **flutter pub get.**

    

    vi. Run app: **flutter run â†’ select device â†’ Output**

    

    when you add something in **pubspec.yaml** run following commands in Terminal:

    

    1. **flutter pub get**

    2. if not working use: **flutter clean** then use **flutter pub get**

6. Update main.dart file : Follow Instructions

    1. import file in main.dart using : ***import* 'question1.dart';** 

    2. In main.dart Remove the Text:(â€™Hello world!â€™) from : **child: Text('Hello World!'),**

    3. Add class name : **child: Question_answer(),**

    4. **Optional : change font, Image path, details etc.**

7. Run flutter app: flutter run
